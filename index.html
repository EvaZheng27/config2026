😌 攤位的人靠搭話吃飯，他們希望你丟問題讓他們講。你只負責開門，讓對方講。 照念下面就好。
🗝️ 萬用 3 句（對任何廠商都能用）

What are you showing at Config this year?

① 開場・讓他講：你們這次來 Config 主打什麼？

I'm a UI designer moving into product / PM in iGaming — how would this help someone like me?

② 拉到自己：我是 UI 設計、正往 PM 走（做 iGaming），這對我這種人有什麼幫助？

What should I check out after the conference? Any promo code?

③ 收尾・帶走價值：結束後我可以去看什麼？有優惠碼嗎？

🆘 腦袋全空白時的救命句

What's the one thing you'd want me to remember about you?

你最希望我記得你們哪一點？（把球完全踢回去）

Oh interesting — why is that? / How does that compare to…?

接話神器：喔有意思，為什麼？／ 那跟○○比起來呢？（再撐 2 分鐘）

🎯 看到什麼人，丟什麼題（應對矩陣）
🔧 工程師 Engineer

What are you building right now — and what's the most annoying part?

你最近在做什麼？最煩的部分是哪裡？（工程師最愛吐槽痛點）

How is AI actually changing your day-to-day?

AI 真的怎麼改變你的日常工作？

🎨 設計師 Designer（同行）

What's your team's biggest design challenge this year?

你們團隊今年最大的設計挑戰是什麼？

What tool or workflow changed your work the most recently?

最近哪個工具/流程最改變你的工作？（互相挖工具）

📋 PM / 產品經理（你的目標，偷學）

How did you get into product? What did you do before?

你怎麼進產品的？之前做什麼？（找轉職路徑）

What does a great PM–designer relationship look like to you?

好的 PM 跟設計師合作，在你眼中長怎樣？

🚀 創辦人 / 新創 Founder

What made you start this? What problem were you living with?

你為什麼創這個？你自己卡在什麼問題？

Who is this really for?

這到底是做給誰用的？（逼出定位，很 PM）

🏢 大廠的人 Big tech（Google / Meta / MS）

How does a team that big keep design consistent?

這麼大的團隊怎麼維持設計一致？

What could a small team learn from how you work?

小團隊能從你們的做法學到什麼？

🎤 講者 Speaker（聽完演講追上去）

Your talk really landed — especially the part about ___. Can I ask…?

你的演講我很有感，尤其 ___ 那段（講具體一點）。想問…

What didn't fit into the talk that you wish you'd said?

有什麼沒講到、但你很想補的？

🪑 鄰座 / 排隊的觀眾 Fellow attendee

What's the best session you've been to so far?

你目前聽過最讚的是哪一場？（最自然的破冰）

What brought you to Config?

什麼原因讓你來 Config？

💼 聊職涯 / 轉職（你正在轉 PM）

I'm a designer moving into PM — any advice for the transition?

我設計轉 PM，這個轉換你有什麼建議？

Looking back, what would you tell someone just starting that move?

回頭看，你會跟剛起步的人說什麼？

🐙 GitHub AI 團隊（Copilot / AI agents）

How are designers, not just devs, using Copilot now?

現在設計師（不只工程師）怎麼用 Copilot？

What can the new Copilot agents do end-to-end today?

現在 AI agent 能一條龍做到哪？

How do PMs at GitHub use Copilot day to day?

（PM 視角）GitHub 的 PM 平常怎麼用 Copilot？

If I'm not an engineer, what's the easiest way to start getting value from Copilot?

不是工程師的話，最簡單的入門用法是什麼？

‹div›RIOTS（html.to.design 等工具）

Do you have a Config promo code or free Pro trial for html.to.design?

html.to.design 有 Config 限定優惠碼／免費 Pro 試用嗎？

Can you run html.to.design on this site and show me the editable result?

能不能用這個網站當場跑一次，看變成可編輯 Figma 的結果？

How do teams keep Figma and code in sync with Backlight / story.to.design?

（設計系統/PM）團隊怎麼讓 Figma 跟程式碼不脫節？

👋 禮貌結束、隨時可走

This was great, I'll check it out — thank you!

聊得很開心，我會去看看，謝謝！（講完就能走，完全 OK）

🧠 心法：你問問題＝你掌握節奏，不用想「我要講什麼厲害的」。讓對方講，你點頭追問就夠了。
